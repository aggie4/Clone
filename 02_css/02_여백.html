margin / padding의 값을 주는 방법

margin이나 padding에 값을 1개 초과해서 주면 값을 주는 만큼 방향으로 여백이 생긴다.

margin / padding : Npx; 사방으로 Npx만큼 여백이 들어온다

margin / padding : Npx Npx; 위/아래 , 왼/오른

margin / padding : Npx Npx Npx; 위 왼/아래 아래

margin / padding : Npx Npx Npx Npx; 위 오 아 왼

너비값이 정해진 요소를 화면성(띄워진 창의 크기에서) 또는 내 부모요소에서 중간정렬을 위한 값이다.

매번 너비가 바뀌므로 정확하게 수치상으로 계산하는 것은 비효율적이다. 반응형의 기초 작업의 일부 , 많이 사용된다

글자만 중간에 보내고 싶다고 박스를 만들지 않고 그냥 사용하면 안된다. 글자 정렬은 다른 속성이므로 무조건 박스를 만들어서 기본 너비값을 넣어주고 사영해야한다.

나중에 작성한 여백값이 우선순위로 적용된다.

위에 여백 값을 주는 방버에서 2개 값을 적어 사용하는 것과 같은 방법으로 사용가능하다. 예를들어) Npx auto 라고 입력하면 위/아래에 Npx 만큼 여백을 주고 좌/우 로는 가운데 정렬된다
