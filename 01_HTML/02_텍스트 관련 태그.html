strong / em / mark / b / small : 사용 빈도가 조금 높은 것들

sup / sub / ins / del : 위에 것들 보다 사용 빈도가 조금 낮은 것들


3. 그나마 많이 쓰는 것들

- strong태그는 글자를 강조, 굵은 글자체로 나타난다.

- em태그는 글자 기울임체로 나타난다.

- mark태그는 노란색 배경이 나타난다.

- b태그는 굵은 글자체로 나타난다.

- small태그는 작은글자체로 나타난다.


4. 조금 덜 쓰는 것들

- sup태그는 윗첨자를 적어준다. 윗첨자 : AB2

- sub태그는 아래첨자를 적어준다. 아래첨자 : AB2

- ins태그는 밑줄을 그어준다. 밑줄

- del태그는 중간줄을 그어준다. 중간줄


위에 태그들은 단독으로도 사용 가능한데, 가급적이면 다른 태그 안에 끼워서 쓰는게 좋다.

위에 태그들은 인라인요소입니다.
